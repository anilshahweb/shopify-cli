How to Download an Existing Shopify Theme Using Shopify CLI?
Shopify store ki existing theme ko local system me download karne ke liye:
Sabse pahle project ke naam se local system me ek folder create kar lenge aur us folder ko VS Code me open kar lenge.
Uske baad VS Code me Terminal open kar lenge.
Uske baad existing theme ko local system me download (pull) karne ke liye shopify theme pull command ka use karenge.
Uske baad agar ek se jyada themes available hongi, to jis theme par kaam karna hai use select kar lenge.
Uske baad selected theme ki files local system me download ho jayengi aur VS Code me show hone lagenge.
