How to Install and Set Up Shopify CLI for Theme Development?
Shopify CLI install aur setup karne ke liye:
Sabse pahle jis Shopify store par kaam karna hai, uska access le lenge. Yeh store Shopify Partner ka development store bhi ho sakta hai aur koi live website bhi ho sakti hai.
Uske baad VS Code ko local system me download aur install kar lenge.
Uske baad Shopify CLI ko local system me download aur install kar lenge. Iske liye system me Node.js install hona chahiye.
Uske baad Node.js install hai ya nahi check karne ke liye Windows PowerShell ya PowerShell ko Run as Administrator mode me open kar lenge.
Uske baad PowerShell me node -v command ka use karenge. Agar version number show ho raha hai, to iska matlab hai ki Node.js successfully install hai.
Uske baad Shopify CLI install karne ke liye npm install -g @shopify/cli@latest command ka use karenge.
Uske baad Shopify CLI successfully install hua hai ya nahi check karne ke liye shopify version command ka use karenge.
Uske baad Shopify account me login karne ke liye shopify auth login command ka use karenge.
Uske baad agar ek se jyada accounts available honge, to jis account se kaam karna hai use select kar lenge aur browser me login process complete karke account ko authorize kar denge.
