How to Start Local Shopify Theme Development Using Shopify CLI?
Shopify theme ko local system par live preview ke saath develop karne ke liye:
Sabse pahle downloaded Shopify theme ko VS Code me open kar lenge.
Uske baad VS Code me Terminal open kar lenge.
Uske baad live preview ke saath development start karne ke liye shopify theme dev command ka use karenge.
Uske baad agar store password protected hai, to store password enter kar denge.
Path: Shopify Admin → Online Store → Preferences → Password Protection
Uske baad browser me ek preview URL open ho jayega.
Uske baad VS Code me kisi bhi file me jo bhi changes karenge aur file save karenge, Shopify CLI automatically changes ko sync kar dega aur preview URL me latest changes show hone lagenge.
